# Club Penguin Full Stack Dev

A fun little full-stack project inspired by the nostalgic world of Club Penguin, built to learn the fundamentals of modern web development.

- Learning how to build a frontend with interactive UI
- Building a backend with Node.js + Express
- Connecting everything through REST APIs
- Working with a MySQL database
- Understanding clean architecture.

Tech Stack

- React (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)
- JWT (Auth)
- bcrypt (Security)
